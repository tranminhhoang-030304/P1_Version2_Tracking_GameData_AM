# Game Analytics Frontend (Tracking Event)

Dự án Frontend Dashboard hiển thị dữ liệu phân tích game, được xây dựng bằng Next.js 15+, React 19, TailwindCSS và Recharts.

## 1. Yêu cầu hệ thống (Prerequisites)
Để chạy được dự án, server cần cài đặt:
- **Node.js**: Phiên bản 20.9.0 trở lên (Khuyên dùng Node 20 LTS).
- **Package Manager**: NPM (đi kèm Node) hoặc Yarn/PNPM.

## 2. Cài đặt (Installation)

```bash
# Cài đặt các thư viện
npm install
# Hoặc nếu dùng yarn
yarn install
